== Description ==

Testimonials Manager is a WordPress plugin that allows you to manage and display testimonials for your blog, product or service. It can be used to build your portfolio or to encourage readers to subscribe / buy your products. Fully customizable with your own CSS and probably the best testimonial plugin for WordPress. 

Just enter new testimonials in the "Add New" screen. You can either display user images from Gravatar.com or upload your own image. Uploaded images are automatically resized to your requirements. 

A new testimonials page will be automatically created for the first time user, or you can also create your own page and paste 
the following code to display your testimonials:

[show_testimonials]

To rotate testimonials in your sidebar, use the testimonials widget.

You can customize the appearance of the testimonials on your testimonials page as well as the sidebar using the Custom CSS box.

For more information refer to: <a href="http://www.profitplugs.com/testimonials-manager-wordpress/" target="_blank">Testimonial Manager Plugin for Wordpress</a>

== Installation ==

1. Download the Plugin ZIP File
2. Upload the Folder testimonials_manager to your wordpress plugin folder
3. Activate the plugin
4. Add your new testimonials 
5. Use the widget to display testimonials on your sidebar
